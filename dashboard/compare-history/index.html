Compare History

Project score drift over time, budgets, and reruns.

One chart per system. One colored line per benchmark. Default view normalizes each benchmark's history so movement is readable even when raw scales differ hard between accuracy and perplexity.

Point radius tracks budget size. Hover points for exact metric, budget, seed, train time, and run id. Raw values stay available in the tooltip even when the chart uses normalized score. Open through a tiny local server, for example python3 -m http.server inside dashboard/compare-history, so the browser can fetch CSV files.
