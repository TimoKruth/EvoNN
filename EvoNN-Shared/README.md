# EvoNN-Shared

Shared substrate package for the EvoNN monorepo.

Current scope:
- compare/export contract models
- benchmark identity and resolution helpers
- normalized budget metadata models
- run manifest helpers
- telemetry envelope models

Non-goals:
- search logic
- genome definitions
- engine-specific runtime/compiler code
- mutation/crossover operators

This package is intended to stay dependency-light and engine-agnostic.
